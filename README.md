# islamic-apps-prayer

##🚀 Hackathon Project – Prayer Times App

Developed during a hackathon, this project is a modern and user-friendly Islamic application that provides accurate daily prayer times based on any location worldwide. The app is designed to help Muslims stay connected with their prayers through a simple, accessible, and reliable interface.

Built using Kotlin, the application dynamically calculates prayer schedules using geographical coordinates and widely accepted calculation methods, ensuring precision for users across different regions.

##✨ Key Features
🌍 Global location support for prayer time calculation
🕌 Accurate timings for Fajr, Dhuhr, Asr, Maghrib, and Isha
⏰ Real-time updates based on user location
📱 Clean and intuitive mobile UI
🔄 Support for multiple calculation methods
